HTML (HyperText Markup Language) is the markup language involved in creating webpages. It is made up of structure-specific tags surrounding content, including text, images, videos, and much much more. HTML markup looks like this: 



<html>
	<head>
		<title>A simple webpage</titlegt;
	</head>
	<body>
		<h1>This is a simple webpage!</h1>
	</body>
</html>

CSS, Cascading Style Sheets, is the way we make HTML webpages look good (like how this page looks right now). It works by specifying something to style (a selector), then applying a value to a certain property to change how the selected element looks. For instance, this CSS makes all paragraphs red: 



p {
	color: red;
}

JavaScript (also known as ECMAScript) is a programming language used to add behavioral components to webpages. It can do pretty much anything if it is coupled with a server-side language (see PHP below), including making games, forms, and even email programs on a website! JavaScript is by far one of the simplest scripting languages to learn - here's something that gives you a popup when the user clicks on a button (coupled with HTML): 



<button id="theButton" onclick="myFunction()">Click me!</button>

function myFunction() {
	alert("You clicked a button!");
}

Note: In one of the most confusing name relationships ever, JavaScript has nothing at all to do with Java.

PHP (Hypertext PreProcessor) is a server-side scripting language that is used for many things, including form handling, emails, and scoreboards for contests. It's by far the simplest server-side language to learn. Below is a simple email program in PHP: 



<?php
$to = "jack@jackbashford.com";
$subject = "PHP Example";
$body = "This is a PHP example!";
mail($to, $subject, $body);
?>
